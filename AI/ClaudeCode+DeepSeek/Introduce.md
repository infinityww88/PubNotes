CC Switch 是一个用于管理和切换 Claude Code、Codex等不同 AI 供应商和 MCP（Model Control Plane）的桌面应用工具。旨在为开发者提供便捷、高效的 API 配置管理体验，支持一键切换、集中管理、多端点及速度测试等功能。

Claude Code和Codex选项功能决定了 列表展示、MCP配置、提供商配置 的类型，即当切换到【Cluade】后添加的MCP和提供商都是对应Claude Code相关的配置

CC Switch提供了 通用、高级 和 关于 功能页面

通用设置包含 语言、外观、窗口行为 设置，其中窗口行为中的【应用到 Claude Code 插件】可以将VS Code等编辑器中的Claude Code扩展供应商进行一键同步

高级功能提供了 CC Switch、Claude Code、Codex 的配置文件目录，以及方便备份迁移的 导入导出配置 功能

提供商配置
点击【添加供应商】会展示供应商配置信息，CC Switch提供了 模版配置 和 自定义配置 2种形式进行配置，默认提供了 DeepSeek、GLM、ModelScope 等众多提供商的模版配置

选择指定提供商会默认填充供应商信息

没有对应的提供商模版，可以在【预设供应商】中选择【自定义配置】进行配置
