# Misc Editing Tools

## Sort Elements

reorder匹配的选择的元素

- View Z/X Axis：沿着View空间Z/X排序
- Cursor Distance：按照距离3D cursor的距离排序
- Material：按照material index排序

## Separate

将mesh的一部分分离为独立的object

- Selected
- All Loose Parts
- By Material

## Mesh Option

- Auto Merge

  当开启Auto Merge，只要一个vertex移动到另一个vertex的threshold设置的阈值之内，它们就自动合并。这个选项只影响交互式操作
