# Introduction

动画是物体随时间移动或改变形状

- 物体作为整体移动
- 动画物体的vertices或control points
- 基于其他物体的运动而移动或改变形状（parent、hook、armature等）

## Rigging（绑定）

Rigging是给物体添加控制的通用术语，通常用于动画目的。Rigging涉及一些功能

- Armatures

  给物体添加灵活的关节，通常用于骨骼动画

- Constraints

  为rig添加运动约束

- Object Modifiers

  Mesh deformation会被大量涉及

- Shape Keys

  面部表情
