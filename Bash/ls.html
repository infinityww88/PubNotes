展示目录内容

描述

对于每个不是目录类型的文件，ls展示文件的信息
对于每个目录类型的文件，ls展示目录内部包含的文件信息
如果给出多个操作数，先展示非目录文件，再展示目录内容。目录和非目录操作数的ls显示按照字母表顺序独立排序 如果没有操作数，ls展示当前目录的文件信息

一些选项相互冲突覆盖，按照从左到右的顺序引用，最右面的决定使用的选项

长格式总是每行一个entry。长格式第一个字符是文件类型字符，包括

  • b: 块文件
  • c：字符文件
  • d：目录
  • l：符号链接
  • s：socket文件
  • p：管道文件
  • -：常规文件

ls默认排序是文件名的字母表顺序(a-z)，-S开启size排序（最大在前），-t开启时间排序（最新在前），因为当我们需要使用size排序时通常是想看大文件，使用time排序时通常是想看最新的文件。-r反转默认顺序，-f关闭排序。

文件时间有两个应用的地方，排序和长格式显示。有4种时间：

  • 文件内容最后修改时间（默认）
  • 文件状态最后修改时间（-c）
  • 文件最后访问时间（-u）
  • 文件创建时间（-U）

排序相关选项：-f, -s, -t, -r
时间相关选项：-c, -U, -u
符号链接相关选项：-H, -L, -P
文件类型相关选项：-F, -p
目录相关选项：-d, -A, -a
文件大小相关选项：-s -S -k -h

选项

  • -1

    1 line. 强制每行显示1个entry。当输出到控制台时，默认ls会被格式化为表格形式，可以通过-1强制显示为1列。当输出到管道时，-1是默认行为

  • -C

    column. 与-1相反，强制以multi-column形式显示

  • -A

    not all. 显示除了.和..之外的所有entries

  • -a

    all. 包含所有以.开头的文件名

  • -c

    change time. 使用文件状态最后修改时间作为sorting(-t)或者long printing(-l)的时间

  • -d

    directory. 对目录类型操作数不再显示目录内容，而是显示目录本身信息

  • -F

    format. 在每个目录文件后面显示/，在每个可执行文件后面显示*，在每个符号连接后面显示@，在每个socket文件后面显示=，在每个whiteout后面显示%，在每个管道后面显示|

  • -p

    path. 在每个目录文件的后面打印/

  • -f

    flat. 输出不是排序的，同时开启-a选项

  • -G

    颜色化输出。等价于在环境变量中设置CLICOLOR

  • -H

    如果操作数是符号链接，则跟踪连接。只在没有-F, -d, -l选项时有效

  • -L

    link. 连续跟踪符号连接直到最终文件。取消-P选项

  • -P

    如果参数是符号链接，显示链接自身而不是被引用的文件。取消-H, -L选项

  • -h

    当使用长格式-l时，使用度量后缀：Byte，Kilobyte，Megabyte，Gigabyte，Terabyte和Petabyte以减少显示的size数字到3个以下，K、M、G、T、P是二进制的

  • -k

    kilo or block. 如果-s选项开启，打印kilobytes大小的size，而不是blocks。

  • -l

    以长格式显示文件信息。如果输出是终端，则所有文件的全部size在列表之前被打印

  • -R

    recursive. 递归展示遇到的子目录

  • -r

    reverse. 反转排序顺序。

  • -S

    sort. 按照文件大小排序

  • -s

    size. 显示每个文件实际使用的blocks，每个block为512字节，取整到下一个整数。如果输出到终端，在list显示之前打印全部文件大小之和

  • -T

    time. 当显示长格式时，显示完整的实际信息，包括month，day，hour，minute，second和year

  • -t

    在按照字面量排序之前先按照文件的mtime（文件内容最后修改时间）排序。ls默认显示的时间就是mtime，使用-c切换为文件状态修改时间，使用-u切换为文件最后访问时间

  • -u

    使用文件最后访问时间作为-t和-l的时间

  • -U

    使用文件创建时间作为-t和-l的时间
